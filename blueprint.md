# 프로젝트 개요
이 프로젝트는 사용자가 옵션을 추가하고 돌릴 수 있는 간단하고 현대적인 돌림판 웹 애플리케이션입니다.

## 구현된 기능 및 디자인
- **현대적인 UI/UX**: 블러 효과(backdrop-filter), 부드러운 애니메이션, 그리고 현대적인 색상 팔레트를 사용합니다.
- **돌림판 로직**: Canvas API를 사용하여 부드러운 회전 애니메이션(ease-out 효과)과 결과 산출 로직을 구현했습니다.
- **옵션 관리**: 사용자가 실시간으로 옵션을 추가하고 돌림판에 반영할 수 있습니다.
- **다크/화이트 모드**: 사용자 선호에 따라 테마를 변경할 수 있으며, 설정은 브라우저의 `localStorage`에 저장되어 유지됩니다.

## 최근 변경 사항 (다크/화이트 모드 추가)
- **CSS 변수 도입**: 색상 및 배경 처리를 위한 CSS 변수(`:root`)를 사용하여 테마 전환을 용이하게 했습니다.
- **테마 전환 버튼**: 우측 상단에 플로팅 버튼을 추가하여 언제든지 테마를 바꿀 수 있습니다.
- **상태 유지**: 사용자가 마지막으로 선택한 테마를 기억합니다.
- **Canvas 연동**: 테마 변경 시 Canvas의 테두리 색상 등이 테마에 맞춰 업데이트되도록 구현했습니다.

## 기술 스택
- HTML5 (Canvas)
- CSS3 (Modern Features: CSS Variables, Flexbox, Transitions)
- JavaScript (ES6 Modules, LocalStorage)
